- 核心包 `me.earthme.luminol.api` 定义线程化区域监控契约（`ThreadedRegion`, `RegionStats`, `TickRegionData`），直接映射底层引擎状态。
- 事件子系统按领域分包：`entity` 处理异步传送与实体门户逻辑（如 `EntityTeleportAsyncEvent`, `PostEntityPortalEvent`），`portal` 处理末地平台与定位逻辑（如 `EndPlatformCreateEvent`）。
- 插件集成层位于 `org.leavesmc.leaves.plugin`，通过 `FeatureManager` 接口和 `Features` 常量类提供特性开关支持。
- 辅助工具包 `gg.pufferfish.pufferfish` 包含 Sentry 上下文注入与 SIMD 硬件检测逻辑，作为底层性能与安全监控支撑。
